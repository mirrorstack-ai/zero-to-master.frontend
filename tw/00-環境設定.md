# Phase 0：環境設定

> 目標：安裝好所有開發工具，確認可以正常使用。

## 1. Node.js 和 pnpm

### macOS

```bash
# 安裝 Homebrew（套件管理工具）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 透過 nvm 安裝 Node.js
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash
nvm install --lts

# 安裝 pnpm
corepack enable
corepack prepare pnpm@latest --activate
```

### Windows

1. 從 https://nodejs.org 下載安裝 Node.js（選 LTS 版本）
   - 或用 [nvm-windows](https://github.com/coreybutler/nvm-windows)：
   ```bash
   nvm install lts
   nvm use lts
   ```

2. 安裝 pnpm：
   ```bash
   corepack enable
   corepack prepare pnpm@latest --activate
   ```

## 2. Git

### macOS

```bash
brew install git
```

### Windows

從 https://git-scm.com/download/win 下載安裝
- 選「Use Git from Git Bash only」
- 選「Checkout as-is, commit Unix-style line endings」
- 建議：安裝 [Windows Terminal](https://apps.microsoft.com/detail/9n0dx20hk701)，把 Git Bash 設為預設

### 設定

```bash
git config --global user.name "你的名字"
git config --global user.email "你的@email.com"
git config --global init.defaultBranch main
```

## 3. GitHub CLI

### macOS

```bash
brew install gh
```

### Windows

```bash
winget install GitHub.cli
```

### 登入

```bash
gh auth login
# 選：GitHub.com → SSH → Yes → Login with a web browser
```

## 4. VS Code

1. 從 https://code.visualstudio.com 下載安裝
2. 安裝這些擴充套件：
   - **ESLint** — 程式碼檢查
   - **Prettier** — 程式碼格式化
   - **Tailwind CSS IntelliSense** — Tailwind 自動補完
   - **Error Lens** — 行內顯示錯誤
   - **GitLens** — Git 歷史紀錄
3. 開啟存檔自動格式化：
   - 設定（`Cmd + ,` / `Ctrl + ,`）→ 搜尋「format on save」→ 打勾

## 5. Docker Desktop

從 https://www.docker.com/products/docker-desktop/ 下載安裝

安裝完確認：

```bash
docker --version   # 27.x
```

## 6. Claude Code

### macOS

```bash
brew install claude-code
```

### Windows

```bash
winget install Anthropic.ClaudeCode
```

### 登入

```bash
claude
# 瀏覽器會跳出來，照著步驟登入就好
```

### 安裝 Everything Claude Code 外掛

```bash
# 在 Claude Code 裡面執行：
/plugin marketplace add affaan-m/everything-claude-code
/plugin install everything-claude-code@everything-claude-code
```

## 確認清單

全部跑一次，確認有正確顯示版本號：

```bash
node -v          # v24.x 或更高
pnpm -v          # 10.x
git --version    # 2.53+
gh --version     # 2.88+
docker --version # 27.x
claude --version # 2.x
code --version   # 1.113+
```

- [ ] 上面的指令都有印出版本號
- [ ] VS Code 可以打開，擴充套件都裝好了
- [ ] 輸入 `claude` 可以正常啟動
